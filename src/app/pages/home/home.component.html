Olá! Seja bem vindo ao sistema $WebFinanças$

O $WebFinanças$ foi desenvolvido por mim Desenvolvedor Full Stack.
Esse é um sistema para gerenciamento de despesas e receitas simples desenvolvido em Angular 12 e Spring Boot 2.
O sistema está rodando na nuvem sendo o front na Netlifi e o back na Heroku.
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT e muito mais. O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.

Fique a vontade para testar o sistema e deixar sua sugestão.
